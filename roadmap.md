# Future Roadmap

Ideas for future development (Offline-first, Audio Masking, etc):

1. **White/Pink Noise Masking**
   - Play masking sounds that help block distractions:
   - `AudioMaskingPlayer` with white/pink noise files.

2. **Task Area Segregation**
   - Work area tasks
   - Study area tasks
   - Personal area tasks
   - Project-based groupings
   - Color-coded categories
   - One-area-at-a-time focus mode

3. **Visual Task Planning**
   - Time-blocking visual calendar
   - Icon-based task representation
   - Color-coded priority levels
   - Drag-and-drop task scheduling
   - Timeline view of daily activities

4. **Brain Dump / Thought Capture**
   - Quick-add floating button for thoughts
   - Voice-to-text task entry
   - Inbox for unsorted tasks
   - Automatic categorization suggestions
   - "Parking lot" for future ideas

5. **Session Statistics & Insights**
   - Focus time per day/week/month
   - Most productive hours heatmap
   - Task completion rate
   - Average session length
   - Distraction frequency tracking
   - Weekly summary reports

6. **End-of-Day Reflection**
   - "What went well today?"
   - "What was challenging?"
   - Tomorrow's top 3 priorities
   - Energy level rating
   - Notes for tomorrow

7. **Widget Support**
   - Home screen timer widget
   - Current task display widget
   - Today's task list widget
   - Streak counter widget
   - Quick-start timer button

8. **Offline-First with Cloud Sync**
   - Full offline functionality
   - Background sync when online (WorkManager/SyncAdapter)
   - Conflict resolution
   - Export to JSON/CSV
   - Backup/restore functionality
